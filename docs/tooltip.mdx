## Required Script

The Tooltip component in david-ai requires JavaScript to handle the tooltip functionality and positioning. The library provides smooth transitions and hover behavior through data attributes, powered by Popper.js for accurate positioning.

### Installation

To use the Tooltip component in your project, you first need to install the `david-ai` library via npm:

<CodePreviewTailwindClasses codeSnippet={`npm install david-ai`}/>

### Basic Usage

<CodePreviewTailwindClasses codeSnippet={
`import { initTooltips } from 'david-ai';

// Initialize tooltip functionality
initTooltips();
`
}/>

### Using with Global Access
If you prefer, you can use the DavidAI global object instead of directly importing initTooltips:

<CodePreviewTailwindClasses codeSnippet={
`import * as DavidAI from 'david-ai';

// Initialize tooltip functionality
DavidAI.initTooltips();
`
}/>

### Using with CDN
You can include david-ai via a CDN and initialize tooltip functionality globally in the browser. Add the following script to your HTML file:

## TypeScript

David AI Tooltip component can be used programmatically with TypeScript support for more control and type safety:

### Programmatic Usage with TypeScript

Learn how to use David AI's Tooltip component with TypeScript for enhanced tooltip control. This component provides programmatic methods to show/hide tooltips, customize content and placement, and configure tooltip behavior with full type safety and customization options.


<CodePreviewTailwindClasses codeSnippet={
`import { Tooltip } from "david-ai";
  import type { TooltipConfig, ITooltip } from "david-ai";

  document.addEventListener("DOMContentLoaded", () => {
  // Trigger elements
  const button: HTMLElement | null = document.getElementById("tooltip-button");
  const showButton: HTMLElement | null = document.getElementById("show-tooltip");
  const hideButton: HTMLElement | null = document.getElementById("hide-tooltip");

  if (!button || !showButton || !hideButton) {
    console.error("Tooltip trigger elements or control buttons not found.");
    return;
  }

  // Initialize tooltip config
  const tooltipConfig: TooltipConfig = {
    content: "This is a tooltip for the button.",
    placement: "bottom", 
    tooltipClass: "bg-stone-800 text-white text-xs rounded-md py-1 px-2 shadow-md z-50",
  };

  const tooltip: ITooltip = new Tooltip(button, tooltipConfig);

  // Example programmatic control
  button.addEventListener("click", async () => {
    await tooltip.showTooltip(); // Programmatically show the tooltip
    setTimeout(() => tooltip.hideTooltip(), 2000); // Auto-hide after 2 seconds
  });

  // External control buttons
  showButton.addEventListener("click", () => {
    tooltip.showTooltip();
  });

  hideButton.addEventListener("click", () => {
    tooltip.hideTooltip();
  });
});`
}/>
