## Required Script

The Stepper component in david-ai requires JavaScript to handle the step functionality. The library provides smooth transitions and navigation behavior through data attributes.

### Installation

To use the Stepper component in your project, you first need to install the `david-ai` library via npm:

<CodePreviewTailwindClasses codeSnippet={`npm install david-ai`}/>

### Basic Usage

<CodePreviewTailwindClasses codeSnippet={
`import { initStepper } from 'david-ai';

// Initialize stepper functionality
initStepper();
`
}/>

### Using with Global Access
If you prefer, you can use the DavidAI global object instead of directly importing initStepper:

<CodePreviewTailwindClasses codeSnippet={
`import * as DavidAI from 'david-ai';

// Initialize stepper functionality
DavidAI.initStepper();
`
}/>

### Using with CDN
You can include david-ai via a CDN and initialize stepper functionality globally in the browser. Add the following script to your HTML file:

<CodePreviewTailwindClasses codeSnippet={`<script src="https://cdn.jsdelivr.net/gh/creativetimofficial/david-ai@1.0.6/packages/dist/david-ai.min.js" defer></script>`}/>

## TypeScript

David AI Stepper component can be used programmatically with TypeScript support for more control and type safety:

### Programmatic Usage with TypeScript

Learn how to use David AI's Stepper component with TypeScript for multi-step navigation control. This component provides programmatic methods to navigate between steps, track progress, and configure the stepper behavior with full type safety and customization options.


<CodePreviewTailwindClasses codeSnippet={
`import { Stepper } from "david-ai";
  import type { StepperConfig, IStepper } from "david-ai";

  document.addEventListener('DOMContentLoaded', () => {
  const container = document.getElementById('stepper-container');
  const stepElements = Array.from(container?.querySelectorAll('[data-dui-step]') || []);
  const contentElements = Array.from(container?.querySelectorAll('[data-dui-step-content]') || []);
  const prevButton = container?.querySelector('[data-dui-stepper-prev]');
  const nextButton = container?.querySelector('[data-dui-stepper-next]');

  if (!stepElements.length || !contentElements.length || !prevButton || !nextButton) {
    console.error('Stepper elements not found or improperly configured.');
    return;
  }

  const initialStep = parseInt(container?.dataset.initialStep || '1', 10);

  // Configure the stepper options
  const stepperConfig: StepperConfig = {
    stepElements: stepElements as HTMLElement[],
    contentElements: contentElements as HTMLElement[],
    prevButton: prevButton as HTMLElement,
    nextButton: nextButton as HTMLElement,
    initialStep,
  };

  // Initialize the Stepper with type safety
  const stepper: IStepper = new Stepper(stepperConfig);

  // Example of using IStepper methods
  // stepper.goToStep(2);
  // stepper.next();
  // stepper.prev();
  // const currentStep = stepper.getCurrentStep();

  // Clean up when the page is unloaded
  window.addEventListener('unload', () => {
    stepper.destroy();
  });
});`
}/>
